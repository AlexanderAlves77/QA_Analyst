# QA Analyst Portfolio

This repository contains a collection of Quality Assurance projects developed to demonstrate skills in test automation, software testing, and QA engineering best practices.

The goal of this repository is to showcase real-world testing scenarios, automation frameworks, and testing strategies used in modern software development.
